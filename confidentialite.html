Données collectées

Quinéo collecte uniquement les données strictement nécessaires à son fonctionnement :

  • Données de session stockées localement sur votre appareil (cartons, scores, préférences)
  • Images de cartons transmises temporairement à notre API d'analyse OCR (non conservées)
  • Données de paiement traitées exclusivement par Stripe (nous n'accédons pas à vos coordonnées bancaires)
Quinéo ne collecte pas de données personnelles d'identification, ne crée pas de compte utilisateur et ne conserve aucune donnée sur nos serveurs en dehors du traitement OCR ponctuel.

Finalités du traitement

  • Fonctionnement de l'application (gestion des cartons et des parties)
  • Analyse OCR des cartons via Google Vision API et Anthropic Claude API
  • Traitement des paiements via Stripe (déblocage premium)

Base légale

Le traitement est fondé sur l'exécution du contrat (fourniture du service) conformément à l'article 6(1)(b) du RGPD.

Partage des données

Vos données peuvent être transmises aux sous-traitants suivants dans le cadre strict du service :

  • Google LLC — API Vision (OCR) — traitement ponctuel des images
  • Anthropic PBC — API Claude (OCR) — traitement ponctuel des images
  • Stripe Inc. — Paiement sécurisé
  • Vercel Inc. — Hébergement de l'application

Aucune donnée n'est vendue, louée ou cédée à des tiers à des fins commerciales.

Conservation des données

Les données de session sont conservées localement sur votre appareil et supprimées à votre initiative (désinstallation ou vidage du cache). Les images transmises pour OCR ne sont pas conservées après traitement.
